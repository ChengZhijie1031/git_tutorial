# git学习

使用的是[廖雪峰][廖雪峰教程]的git教程



## 创建版本库

1. 首先新目录下创建git仓库（repository）

   > 注意：windows系统下无法追踪Word文件改动

   使用`git init`初始化命令

   

2. 添加文件到git仓库

   使用`git add `命令添加文件到仓库

   使用`git commit -m "说明文本"`提交文件到仓库

   可以多次`git add`，最终一并提交`git commit`
